## Documentation
https://playwright.dev/

## Environment
* Node 18
* VS Code


Put your GitHub credential in VS Code terminal.integrated.env.* settings for you OS, for example
```json
{
  "terminal.integrated.env.osx": {
    "GITHUB_USER": "user1",
    "GITHUB_PASSWORD": "Pazzwerd!"
  },
}
```
## Install
```console
yarn
```
## REPL
```console
yarn repl
```
## Run tests
```console
yarn test
```
